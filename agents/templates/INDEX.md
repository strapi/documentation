# Templates Index (for agents)

This non-docs index lists the canonical authoring templates for CMS documentation. It is meant for agents and contributors browsing the repo, not for Docusaurus rendering.

## Feature Template
- Purpose: Structure for feature pages (TL;DR, intro, identity card, configuration, usage) to keep content consistent and task‑oriented.
- Path: `agents/templates/feature-template.md`

## Plugin Template
- Purpose: Standard layout for plugins with identity details, install steps, configuration (admin UI and code), and usage tasks.
- Path: `agents/templates/plugin-template.md`

## Configuration Template
- Purpose: Clear guide for configuration topics, including location, available options, env variables, and per‑environment overrides.
- Path: `agents/templates/configuration-template.md`

## Guide Template
- Purpose: Step‑by‑step how‑to structure with prerequisites, numbered steps, and validation.
- Path: `agents/templates/guide-template.md`

## API Template
- Purpose: Lightweight API page skeleton covering endpoints, auth, parameters, and example requests/responses.
- Path: `agents/templates/api-template.md`

## Migration Template
- Purpose: Migration/breaking‑change format with impact, before/after, and a numbered migration sequence.
- Path: `agents/templates/migration-template.md`
