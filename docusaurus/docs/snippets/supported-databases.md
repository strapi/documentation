| Database   | Recommended | Minimum |
|------------|-------------|---------|
| MySQL      | 8.4         | 8.0     |
| MariaDB    | 11.4        | 10.3    |
| PostgreSQL | 17.0        | 14.0    |
| SQLite     | 3           | 3       |

Strapi does not support MongoDB (or any NoSQL databases), nor does it support any "Cloud Native" databases (e.g., Amazon Aurora, Google Cloud SQL, etc.).
