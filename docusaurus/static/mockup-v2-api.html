Entries

CRUD operations on your content-type entries. Strapi auto-generates these endpoints from your content-type schemas.

Base URL http://localhost:1337
GET /api/:pluralApiId

List entries

Returns a paginated list of entries. Supports filtering, sorting, field selection, population of relations, and locale-based queries.

Query Parameters
sort
string | string[]
optional
Sort by field. Use field:asc or field:desc
filters
object
optional
Filter with operators like $eq, $contains, $gt
populate
string | object
optional
Relations and components to include in the response
pagination[page]
integer
optional
Page number. Default: 1
pagination[pageSize]
integer
optional
Items per page. Default: 25, max: 100
fields
string[]
optional
Select specific fields to return
cURL JavaScript Python
GET /api/restaurants 
curl 'http://localhost:1337/api/restaurants' \
  -H 'Authorization: Bearer <token>' \
  -G \
  --data-urlencode 'sort=name:asc' \
  --data-urlencode 'pagination[page]=1' \
  --data-urlencode 'pagination[pageSize]=25' \
  --data-urlencode 'populate=*'
Response 200 OK 
{
  "data": [
    {
      "id": 1,
      "documentId": "a1b2c3d4e5",
      "name": "Biscotte Restaurant",
      "description": "A cozy place",
      "createdAt": "2026-01-15T09:00:00.000Z",
      "updatedAt": "2026-01-15T09:00:00.000Z"
    }
  ],
  "meta": {
    "pagination": {
      "page": 1,
      "pageSize": 25,
      "pageCount": 1,
      "total": 1
    }
  }
}
POST /api/:pluralApiId

Create an entry

Creates a new entry for the collection type. Send field values inside a data object in the request body.

Body Parameters
data
object
required
Object containing the field values to set
data.name
string
required
Name of the restaurant
data.description
richtext
optional
Description in rich text format
data.rating
integer
optional
Rating from 1 to 5
cURL JavaScript Python
POST /api/restaurants 
curl -X POST \
  'http://localhost:1337/api/restaurants' \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "data": {
      "name": "New Restaurant",
      "description": "A great place",
      "rating": 5
    }
  }'
Response 201 Created 
{
  "data": {
    "id": 2,
    "documentId": "f6g7h8i9j0",
    "name": "New Restaurant",
    "description": "A great place",
    "rating": 5,
    "createdAt": "2026-04-04T14:30:00.000Z",
    "updatedAt": "2026-04-04T14:30:00.000Z"
  },
  "meta": {}
}
DELETE /api/:pluralApiId/:documentId

Delete an entry

Permanently removes an entry by its document ID. This action is irreversible.

Path Parameters
pluralApiId
string
required
Plural API ID of the content type (e.g. restaurants)
documentId
string
required
The document ID of the entry to delete
cURL JavaScript
DEL /api/restaurants/a1b2c3d4e5 
curl -X DELETE \
  'http://localhost:1337/api/restaurants/a1b2c3d4e5' \
  -H 'Authorization: Bearer <token>'
Response 200 OK 
{
  "data": {
    "id": 1,
    "documentId": "a1b2c3d4e5",
    "name": "Biscotte Restaurant",
    "description": "A cozy place"
  },
  "meta": {}
}
