REST API Endpoints

Use the REST API to interact with your content-types. Strapi automatically generates API endpoints for your content types.

GET Get entries

Returns entries matching the query filters. Use this endpoint to fetch a list of entries from a collection type, with optional filtering, sorting, and pagination.

Parameter Status Description
sort
string | string[]
 Optional Sort the response by field(s)
filters
object
 Optional Filter the results using operators
populate
string | object
 Optional Relations, media fields, components to populate
pagination[page]
integer
 Optional Page number (default: 1)
pagination[pageSize]
integer
 Optional Page size (default: 25, max: 100)
cURL JavaScript Python
GET /api/restaurants 
curl 'http://localhost:1337/api/restaurants' \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  --data-urlencode 'sort=name:asc' \
  --data-urlencode 'pagination[page]=1' \
  --data-urlencode 'pagination[pageSize]=25'
Response
200 OK 
{
  "data": [
    {
      "id": 1,
      "documentId": "a1b2c3d4e5f6",
      "name": "Biscotte Restaurant",
      "description": "A cozy place for brunch",
      "createdAt": "2024-01-15T09:00:00.000Z",
      "updatedAt": "2024-01-15T09:00:00.000Z"
    }
  ],
  "meta": {
    "pagination": {
      "page": 1,
      "pageSize": 25,
      "pageCount": 1,
      "total": 1
    }
  }
}

POST Create an entry

Creates a new entry for the given collection type. The request body should contain a data object with the field values.

Body Parameter Status Description
data
object
 Required Object containing the fields to create
data.name
string
 Required Name of the restaurant
data.description
string
 Optional Description in rich text format
cURL JavaScript Python
POST /api/restaurants 
curl -X POST 'http://localhost:1337/api/restaurants' \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "data": {
      "name": "New Restaurant",
      "description": "A great new place"
    }
  }'
Response
200 Created 
{
  "data": {
    "id": 2,
    "documentId": "g7h8i9j0k1l2",
    "name": "New Restaurant",
    "description": "A great new place",
    "createdAt": "2024-03-20T14:30:00.000Z",
    "updatedAt": "2024-03-20T14:30:00.000Z"
  },
  "meta": {}
}

DELETE Delete an entry

Deletes an entry by its document ID. This action is irreversible and will permanently remove the entry and its associated data.

Path Parameter Status Description
documentId
string
 Required The document ID of the entry to delete
cURL JavaScript
DELETE /api/restaurants/:documentId 
curl -X DELETE \
  'http://localhost:1337/api/restaurants/a1b2c3d4e5f6' \
  -H 'Authorization: Bearer <token>'
Response
200 OK 
{
  "data": {
    "id": 1,
    "documentId": "a1b2c3d4e5f6",
    "name": "Biscotte Restaurant",
    "description": "A cozy place for brunch"
  },
  "meta": {}
}
